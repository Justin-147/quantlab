# Safety Boundary

QuantLab does not connect to brokers, submit real-money orders, store API keys, scrape paid data, or provide investment advice.

V1 and V0.2 support only:

- local CSV data loading;
- synthetic sample data;
- historical backtesting;
- local paper simulation;
- Markdown, HTML, JSON, and chart reporting;
- Streamlit dashboard exploration.

The project is not a trading bot. It does not recommend buying or selling securities. Past performance does not guarantee future results.
